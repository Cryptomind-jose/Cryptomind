# CryptoMind

CryptoMind es un portal estilo blog enfocado en **estrategias y análisis de criptomonedas**, con secciones educativas y un área exclusiva para miembros VIP.

## 🚀 Características principales
- **Inicio**: Presentación clara y profesional.
- **Blog**: Artículos destacados sobre trading y criptomonedas.
- **Señales**: Resultados recientes y rendimiento semanal.
- **Guías**: Recursos para principiantes.
- **VIP**: Planes premium (acceso solo desde el menú).
- **Nosotros**: Información sobre la misión y visión de CryptoMind.

## 🛠 Cómo desplegar en GitHub Pages
1. Clona o sube este repositorio a tu cuenta de GitHub.
2. Ve a **Settings → Pages**.
3. En la sección *Source*, selecciona la rama `main` y carpeta raíz `/root`.
4. Guarda los cambios.
5. Tu sitio estará disponible en:  
   `https://tuusuario.github.io/Cryptomind/`

## 🎨 Personalización
- Cambia el archivo `style.css` para ajustar colores, tipografía y diseño.
- Edita `index.html` para añadir nuevos artículos o secciones.
- Usa `script.js` para controlar qué secciones se muestran al hacer clic en el menú.
